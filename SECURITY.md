# Security Policy

## Supported Versions

Only the latest version of OneDrive Smart Transfer is supported for security updates.

## Reporting a Vulnerability

If you find a security vulnerability, please do NOT report it via a public issue. Instead, please report it via the email below.

**Email**: [himeshmaniyar2017@gmail.com](mailto:himeshmaniyar2017@gmail.com)

We will respond to your report within 48-72 hours and provide an estimate for when the issue will be fixed. We appreciate your help in keeping OneDrive Smart Transfer secure!
